## Features

- สมาชิก เข้าผ่าน LINE Login ได้
- มีสองช่องทางติดต่อ คือ www + LINE OA ของบริษัท
- Web บริษัทมี Landing Page + Blog + SEO (แนว WordPress) + AI หาเจอ
- สมาชิกกดทาง Web และ LINE เพื่อกรอกแบบฟอร์มมา จอง/สั่งซื้อ (ไม่มีเรื่อง Payment)
- มีหน้าหลังบ้าน Admin Dashboard
